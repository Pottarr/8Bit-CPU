# ---------- Code Format -----------
# LD    =>  0000   load to register
# MOV   =>  0001   move value
# ST    =>  0010   store data
# RD    =>  0011   read data


# ADD   =>  0100   add 2 data
# SUB   =>  0101   subtract 2 data
# MUL   =>  0110   multiply 2 data


# WR    =>  0111   write register to output


# AND   =>  1000   and 2 data
# OR    =>  1001   or 2 data
# XOR   =>  1010   xor 2 data
# NOT   =>  1011   not a single data

# BC    =>  1100   branch if carry out
# BZ    =>  1101   branch if zero
# BNZ   =>  1110   branch if not zero
# B     =>  1111   unconditional branch

# -------------- Rule --------------
# 19 bits instruction
# bit 18     : immediate mode    true / false
# bit [17:14] : opcode
# bit [13:11] : register destination
# bit [10:8]  : register operand 1 (ignore if not used)
# based on bit 18
# case 0: bit [2:0]   : register operand 2 ([7:3] will be ignore)
# case 1: bit [7:0]   : 8 bits of binary immediate
